Тестовое Задание от 7Ветров

ЧПУ ссылки. Реализовать на Yii2 функционал, благодаря которому все контентные 

страницы сайта будут иметь свой ЧПУ.

I. Реализация модели ЧПУ в соответствии со следующими требованиями:

1. Набор допустимых символов (a-z0-9), длина — не более 255 символов;

2. Чувствительность к регистру;

3. Сохранение истории для страниц (запрос по старому урлу приводит к 

редиректу на новый урл страницы);

4. Хранение в БД MySQL.

II. Реализация CRUD для модели контентой страницы, связанной с ЧПУ.

III. Обеспечить возможность доступа к экземпляру модели ЧПУ из контроллеров 

Примечание. Структура модели контентной страницы (и контроллер на фронтенде) 

— на своё усмотрение (максимально просто — только для целей тестирования ЧПУ).

Результат выполнения:

	 Исходный код;

	 Миграции, необходимые для тестирования на пустой БД;
